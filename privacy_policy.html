Privacy Policy

Last Updated: April 7, 2026

🔒 Summary: We collect minimal data required for TikTok API integration. No personal data is stored beyond what TikTok provides. You retain full control.

1. Information We Collect

The Agent collects minimal information necessary for operation:

Data Type Purpose Storage
TikTok API Access Token Authentication for posting content Encrypted, temporary (2 hours)
Basic account info (username) Identify account for posting Not stored persistently
Content generation logs Service monitoring and debugging 7 days, then deleted
Posting statistics Performance analytics Anonymous, aggregated

What we DO NOT collect:

  • Personal identification information
  • Email addresses (beyond TikTok account email)
  • Phone numbers
  • Location data
  • Device information
  • Browsing history

2. How We Use Information

Information is used solely for:

  • Authentication: Connecting to TikTok API
  • Content Posting: Uploading videos to your account
  • Basic Analytics: Tracking post count and engagement
  • Service Improvement: Monitoring agent performance
  • Error Handling: Debugging technical issues

3. Data Storage & Security

Storage:

  • TikTok API tokens: Encrypted, valid for 2 hours maximum
  • Logs: Stored for 7 days, then automatically deleted
  • No databases or persistent storage of personal data

Security Measures:

  • Encryption of all API credentials
  • Secure server environment
  • Regular security updates
  • No sharing of data with third parties

4. Third-Party Services

The Agent integrates with:

  • TikTok API: For content posting and account info
  • ProtonMail: For email verification (read-only access)

We do not use analytics services, advertising networks, or social media widgets.

5. Your Rights

You have the right to:

  • Access: Request what data we have (minimal)
  • Deletion: Request deletion of any stored data
  • Revocation: Revoke TikTok API access at any time
  • Control: Stop the service immediately

6. Children's Privacy

This service is not intended for users under 13. We do not knowingly collect data from children.

7. Changes to This Policy

We may update this policy. Continued use of the Agent constitutes acceptance of changes.
